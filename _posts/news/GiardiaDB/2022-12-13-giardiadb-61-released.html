New data in {{project}} 61

Structure Predictions

  • AlphaFold Predictions – are updated to AlphaFold Release 4 (July 2022) which offers predictions for significantly more proteins than Release 3. Use {{project}}'s Genes with AlphaFold Predictions search to explore genes with AlphaFold predictions or visit the Structure Analysis section of your favorite gene page to view data relevant to you.

New features in {{project}} 61

Associated Data Sets

  • Data sets that come from the same publication are now linked. The new 'Associated Data Sets' section on any data set record page will link to other data from the same publication within {{project}} as well as data from organisms supported in different VEuPathDB sites. View tutorial GIF.
