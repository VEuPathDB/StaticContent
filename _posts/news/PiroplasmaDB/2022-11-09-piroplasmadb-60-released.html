New data in {{project}} 60

Single cell RNA-Seq

This data category is new in {{project}} 60. Analyzed data was obtained from the data provider and integrated into {{project}} without further analysis. Users can search for genes reported in a particular data set (Genes by Single Cell RNA-Seq Evidence), view a snapshot of the data on gene record pages (example), and explore the full data set in the cellxgene application linked in the Single Cell RNA-Seq table on gene record pages.

  • Comparative single cell atlases of Babesia species: B. bigemina – Comparative single cell atlases of Babesia species in different hosts. Rezveni et al. bioRxiv preprint
  • Comparative single cell atlases of Babesia species: B. bovis – Comparative single cell atlases of Babesia species in different hosts. Rezveni et al. bioRxiv preprint

New features in {{project}} 60

{{project}} site search accesses Apollo data

  • Gene annotation updates entered in {{project}} Apollo are now indexed and available to the site search. Genes with Apollo annotations that contain your search term will be available in site search results under >Genes >Filter gene fields.

Gene page navigation improvements

  • The RNA-Seq Transcription Summary graph now links directly to the Transcript Expression table. Click a data set title in the RNA-Seq Transcription Summary graph to navigate directly to the details for the same data set in the Transcript Expression table.
  • The gene page content navigation tool now contains links to every table or attribute on the gene page. Open each numbered data section to reveal links to all data within the section. Or use the 'expand all' link at the top of the content tool to view all links.

Single cell RNA-seq Βeta version

  • Single cell RNA-seq data is now represented in {{project}} via UMAP in cellxgene. (Interface options tutorial.) More data and further feature development are expected in coming releases. Please Contact Us with questions or suggestions.
  • A new search, Single Cell RNA-Seq Evidence, returns all genes represented in a scRNA-seq data set. The search is available under Search For, Genes, Transcriptomics.
  • A new table, Single Cell RNA-Seq (cellxgene), appears in the Transcriptomics section of record pages for genes with scRNA-Seq data. Table rows represent individual data sets. Expand the rows to reveal static cellxgene images and links to view the data dynamically in the cellxgene application.

AlphaFold data available

  • Protein structure predictions from AlphaFold are now available in {{project}}. The AlphaFold database (Version 3) was downloaded and structures were associated to {{project}} genes via the UniProt ID. Jumper et al. 2021 and Varadi et al. 2022
  • The AlphaFold search returns all genes with AlphaFold predictions in {{project}}.
  • Gene pages include a table of AlphaFold data and the AlphaFold 3D viewer for dynamic visualization of the predicted structure.
