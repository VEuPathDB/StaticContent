New data in {{project}} 65

New features in {{project}} 65

  • BUSCO scores are now available for annotated genomes in {{project}}. BUSCO estimates the completeness and redundancy of processed genomic data based on universal single-copy orthologs. Scores on VEuPathDB were generated with the Ensemble pipeline which computes scores based on the docker provided by BUSCO itself. BUSCO on PiroplasmaDB GIF.
  • VEuPathDB Galaxy now offers a preconfigured ChIP Seq workflow. This workflow processes ChIP Seq FASTQ files by aligning to your chosen reference genome with BWA then calling peaks using MACS2. Peaks calculated from the input FASTQ files are converted to bigWig format which can be exported from Galaxy to your VEuPathDB account.
  • Strategy Improvement – A new option in the gene result table provides a view of search specific data columns for your preferred subset of genes. Click the 'Show only the Genes in my basket' option from any gene result to filter the result based on the genes in your basket. This can be used in combination with the 'Send to Basket (replace)' option to view search specific data columns for multi-step strategy results. Filter by Basket tutorial.


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary. Changes in annotation and new experimental data may slightly alter your search results by increasing or decreasing the number of hits.
