New data in {{project}} 63

RNA Sequence

New features in {{project}} 63

  • Site search 'Reference Only' organism filter – Large results (> 50,000 hits) from the site search are common since it accesses all {{project}} data/page categories (genes, data types, data sets, organisms, etc.). The new 'Reference Only' toggle offers a one-click filter to reduce the result from all organisms to include only results from VEuPathDB reference orgnanisms.
  • Search details and data sets – Details for the search and the data sets accessed by the search are now available in tabbed pages from the top of the search pages. Previous versions of {{project}} presented this information at the bottom of the page where it was easily missed.
  • Scroll and zoom on gene page browser frames – The dynamic browser views on gene pages now provide active scroll and zoom functions with a reset button to return to the default view. Dynamic browser views are available for Synteny, Alignments, SNPs, Transcriptomics Coverage, Protein Features and Proteomics.


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary.
