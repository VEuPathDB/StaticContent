New features in PlasmoDB 41

Introducing Clustal Omega


EuPathDB sites are now using Clustal Omega for gene page protein alignments in the 'Orthologs and Paralogs within EuPathDB' table and the 'Retrieve multiple sequence alignment or multi-FASTA' tool in the 'Orthology and synteny' section of gene pages. The 'Isolate Alignments in this Gene Region' tool still uses ClustalW.

New Gene page tools

  • Interpro —
    The 'Protein features and properties' section now includes an option to submit the protein sequence for sequence analysis and classification via InterPro.
  • GPI anchor prediction —
    The 'Protein features and properties' section now includes an option to submit the protein sequence for GPI modification site prediction via the GPI Lipid Anchor Project.
