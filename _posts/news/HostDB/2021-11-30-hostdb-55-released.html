New features in {{project}} 55

  • My Organism Preferences is a new option for filtering menus to a subset of organisms in {{project}}. Use the My Organism Preferences page to choose the organisms you are interested in. {{project}} will function as if the database contained only your chosen organisms. The feature is easily disabled/enabled at any time from a button located under the header on every page. More Details
  • The Reference sequence track in JBrowse and PBrowse has a new color scheme. Blah Blah
  • Galaxy Version updated on 8 Oct, 2021: This updated version of the Galaxy software improves job and workflow performance. In addition to the previous set of tools and workflows, we now offer additional tools, such as those for scRNA-seq (Scanpy, Seurat), species ID (CryptoGenotyper), table manipulation (Datamash), and proteomics (MSstats, Search GUI, Peptide Shaker, MaxQuant). Certain tools have been upgraded and now function without errors: bamCoverage, bgzip, tabix and Upload File.


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary. Changes in annotation and new experimental data may slightly alter your search results by increasing or decreasing the number of hits.
