Function prediction

  • CalCENv1 co-expression network UMAP clusters — CalCENv1 co-expression network was projected to two dimensions using UMAP and 18 clusters were identified and annotated through gene set enrichment analysis. O'Meara and O'Meara, 2021.

Phenotype data

  • Phenotypes from PHI-base database — The pathogen – host interactions (PHI) phenotypes and pathogen-specific phenotypes from PHI-base were integrated for 61 different genomes available in FungiDB in this release. This dataset is current with PHI-base version 4.12. We have integrated 5,381 phenotypic annotations mapping to 3,140 genes in total. PHI-base is a web-accessible database that catalogues experimentally verified pathogenicity, virulence and effector genes from fungal, oomycete and bacterial pathogens, which infect animal, plant, fungal and insect hosts. The original data file from PHI-base can be found on PHI-base GitHub site.

New features in {{project}} 57

BLAST Tool Improvements

  • The BLAST (multi-query capable) tool, known previously as BLASTbeta, can perform queries with one or more FASTA-formatted sequences and offers many download types. The tool uses NCBI BLAST to find genes, genomic sequences, ESTs, or PopSet Sequences that have BLAST similarity to your input sequence. BLAST (multi-query capable) is available from the tools menu.
  • A new download option, 'This table (csv)', for the results of multi-sequence queries (combined result) retrieves the {{project}} result table for the results of multi-sequence queries (combined result).
  • View tutorial GIF.

Gene page sequence highlighting

  • Genomic and mRNA sequences on gene pages have a new color scheme. Introns are highlighted in yellow and UTR sequence are highlighted in pink. Previously they were gray and blue, respectively. Example


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary. Changes in annotation and new experimental data may slightly alter your search results by increasing or decreasing the number of hits. When search parameters change with a new release, we invalidate (Ø) the search and ask you to rerun it. When IDs are updated or removed, we map the old IDs to the new ones, remove the old IDs from your Basket, and leave your Favorites page alone.
