New features in {{project}} 63

  • Download Data Files BETA – {{project}} has a new tool for downloading genome scale files such as genome.fasta or GFF files. Similar to other {{project}} tools, all data and files are available by default. To limit the result, choose a value (right) for one or more categories (left). From the filtered list, download a single file by clicking the file name, or multiple files using the check boxes and 'Dowload selected files' button. View Download Beta tutorial GIF.
