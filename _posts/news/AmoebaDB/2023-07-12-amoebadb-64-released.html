New features in {{project}} 64

  • VEuPathDB Galaxy now offers a preconfigured ChIP Seq workflow. This workflow processes ChIP Seq FASTQ files by aligning to your chosen reference genome with BWA then calling peaks using MACS2. Input a pair of treated and control FASTQ data files. Peaks are converted to bigWig format. Coverage files are created in BigWig format for each input file. In addition, a differential coverage track is created in BigWig format with BamCompare. BigWig files can be exported from Galaxy to your VEuPathDB account.
  • Strategy Improvement – A new option in the gene result table provides a view of search specific data columns for your preferred subset of genes. Click the 'Show only the Genes in my basket' option from any gene result to filter the result based on the genes in your basket. This can be used in combination with the 'Send to Basket (replace)' option to view search specific data columns for multi-step strategy results. See Filter by Basket GIF TODO.
