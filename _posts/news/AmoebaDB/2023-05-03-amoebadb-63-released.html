New data in {{project}} 63

Genome Annotation:

  • Naegleria fowleri ATCC 30863 – Community functional annotations for 14 genes were transferred from Apollo to the AmoebaDB genome annotation file.

Proteomics

  • Acanthamoeba castellanii Neff proteome during encystation – Proteomics of encystation in A. castellani Neff at 0, 1, 4, and 8 hours in encysting conditions with controls. Bernard et al. 2022.

New features in {{project}} 63

  • Download Data Files BETA – {{project}} has a new tool for downloading genome scale files such as genome.fasta or GFF files. Similar to other {{project}} tools, all files are available by default. To limit the result, choose a value (right) for one or more categories (left). From the filtered list, download a single file by clicking the file name, or multiple files using the check boxes and 'Dowload selected files' button. Tutorial GIF
