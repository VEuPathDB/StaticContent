New data in {{project}} 63

Proteomics

  • Oocyst wall proteome – A comprehensive analysis of the proteome of oocyst wall of Cryptosporidium parvum was performed using label-free qualitative high-performance liquid chromatography (HPLC) fractionation and mass spectrometry-based qualitative proteomics technologies. Wang et al. 2022

Genome sequence

  • Cryptosporidium parvum 2022 – Fully resolved assembly based on Oxford Nanopore reads and improved using Illumina reads. Menon et al. 2022

Annotation updates

  • Cryptosporidium parvum IOWA-ATCC – This annotated genome was updated to the current INSDC version. GCA_015245375.1

New features in {{project}} 63

  • Download Data Files BETA – {{project}} has a new tool for downloading genome scale files such as genome.fasta or GFF files. Similar to other {{project}} tools, all data and files are available by default. To limit the result, choose a value (right) for one or more categories (left). From the filtered list, download a single file by clicking the file name, or multiple files using the check boxes and 'Dowload selected files' button. Download Beta tutorial GIF.
