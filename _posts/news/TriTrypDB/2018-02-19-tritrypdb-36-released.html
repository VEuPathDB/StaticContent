We are pleased to announce the release of TriTrypDB 36

New data in TriTrypDB 36

Genomes

  • Trypanosoma theileri isolate Edinburgh — This genome sequence and annotation appears for the first time in TriTrypDB. Kelly et al.
  • Trypanosoma cruzi cruzi strain Dm28c — This genome sequence and annotation appears for the first time in TriTrypDB and is distinct from Trypanosoma cruzi Dm28c. The genome sequence and annotation for Trypanosoma cruzi cruzi strain Dm28c is provided by the Probst lab, Instituto Carlos Chagas - ICC/Fiocruz-PR. NCBI Bioproject page.
  • Trypanosoma cruzi Sylvio X10/1-2012 — As requested by the TriTrypDB community, the 2012 version of T. cruzi Sylvio is added back to TriTrypDB and was downloaded from Genbank. An updated version of T. cruzi Sylvio that was downloaded directly from the Andersson lab in 2017 also appears in TriTrypDB. This record is: Trypanosoma cruzi Sylvio X10/1 — Franzen et al.

Quantitative Proteomics

  • BSF and PCF Surface Proteomics — Trypanosoma brucei proteomics of the cell surface from both PCF and BSF stages Shimogawa et al.

New features in this release

Search Page Improvements: search pages were redesigned to include an organism filter and provide better access to help information.

  • Search parameters are now left justified. The Get Answer button remains center justified below the parameters. Parameter help icons, which offer information concerning the parameters when hovered over, now appear before the parameter name instead of after.
  • The Organism Parameter, the first parameter on search pages, has a new look and now contains a filter box for quickly finding your organism of interest. Type any portion of an organism name (at leaset two letters of the genus, species, or strain) in the filter box and the tree collapses to show only those organisms containing your entry. To return to the full tree, click the X in the filter box.

New Paralog Count search

  • The new Paralog Count search returns genes based on the number of assigned paralogs. Find genes that have a number of paralogs within the specified range. Paralogs are defined as genes from the same species that belong to the same OrthoMCL group.
  • Find the search under the Orthology and Synteny category.

Gene Record Page Improvements:

  • A new GO Slim table now appears above the GO Term table on the record pages of genes with GO associated data. The GO Slim table maps GO Terms form the GO Term table to their GO Slim terms based on the Generic GO slim developed by GO Consortium.
  • Column headers in gene page tables now show a help icon ( ) when additional information or definitions are available for the column. Hover over the help icons to reveal additional information.
  • Metabolic pathway and reaction tables on gene pages now include reaction compound names, with CHEBI IDs on mouse over, and a column that indicates if the EC number associated with a gene is an exact match to the EC number in metabolic pathway.

Search Strategy Improvement: new option for combining steps in a search strategy.

  • Search strategies now offer an option to 'ignore' a step when editing a multi-step strategy. To remove a search from the strategy without deleting the step, click Edit for the combine graphic of the step you are interested in ignoring. Then choose 'IGNORE step #' from the Revise Operation popup. For more information see this Ignore Step help document.

Introducing ClinEpiDB, a new member of the EuPathDB family of databases:

  • We are excited to announce the launch of ClinEpiDB, which hosts clinical and epidemiological data for the infectious disease research community. The current release contains data from the PRISM study on malaria in Uganda. Check out this video to learn more about our newest EuPathDB resource!
