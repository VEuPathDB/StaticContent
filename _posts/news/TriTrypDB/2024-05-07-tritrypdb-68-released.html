New data in {{project}} 68

Updated Genomes

  • Leishmania braziliensis MHOM/BR/75/M2903 – This annotated genome was reloaded to reflect changes at the INSDC repository. GCA_000340355.2.

New features in {{project}} 68

  • Long gene product descriptions are now trucated in relation to the width of your browser window. The full product description is available on mouseover and individual transcript product descriptions are available in the All Product Descriptions table on gene pages. Truncated descriptions GIF.
  • The Orthologs and Paralogs table on {{project}} gene pages also serves as a Clustal Omega tool for retrieving alignments between orthologs. This table now includes 'Selection Count' and 'Selection Clear' options for improved tool function. Clustal Omega Counts GIF.
  • The Send To tool is now offered in VEuPathDB.org Strategies. Results created in VEuPathDB.org Strategies can now be ported to {{project}} My Strategies for further investigation. Only genes present in {{project}} species will be included in the ported strategy results. VEuPathDB.org Send To GIF.
  • Newly sorted strategy result tables will now default to page 1 – Search result tables can be sorted based on any column of data. When a table is sorted while viewing page 3, users are returned to page 1 to view the sorted data table.


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary. Changes in annotation and new experimental data September slightly alter your search results by increasing or decreasing the number of hits.
