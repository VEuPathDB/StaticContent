New features in {{project}} 68

  • The Sequence Retrieval Tool offers more input options such as uploading from a text file or URL. New options for configuring or choosing the type of sequence returned are also available.
  • The Product Descriptions table on gene pages is now called All Product Descriptions and contains product descriptions from all relevant sources. The most commonly used product description also appears at the top of the gene page.
  • The Text Search 'Fields' parameter values have changed. Choose 'Product descriptions (all)' to search the All Product Descriptions table on gene pages. Choose 'Product descriptions' to search only the commonly used product descriptions that appear at the top of gene pages.
  • The Organisms parameter in the Identify Genes based on SNP Characteristics search has a new look. Choose your organism of interest to load the Set of Samples parameter with that organism's WGS isolates.
  • Compressed files in the Download Data Files tool are now in .gz format.


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary. Changes in annotation and new experimental data September slightly alter your search results by increasing or decreasing the number of hits.
