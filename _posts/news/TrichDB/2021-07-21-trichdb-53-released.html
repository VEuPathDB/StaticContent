New features in {{project}} 53

  • Genome data set records ({{project}} genome data sets) are updated to increase clarity and reduce redundancy of information on the page. For example, a new table "Explore other Categories of Data Associated with this Genome" contains links to all functional data sets for the organism. The new "Data Context" section provides counts for genes, assembled sequences and transcripts. Functional data set records have received similar updates.
  • ID searches such as the "site search" or "Identify Genes by ID" have been updated to find versions of an ID (e.g. ID.1, ID.2). Specifically, searching for the ID will retrieve ID, ID.1 and ID.2, etc, while searching for ID.1 will retrieve only ID.1
  • The community annotation software, Apollo, was updated to version 2.6.5 (About Apollo.
  • The BLAT sequence search tool is integrated into Apollo. Use the 'Search' tab from the annotator panel to find your sequence of interest in our Apollo-available genomes. (ApolloBLAT.png)
