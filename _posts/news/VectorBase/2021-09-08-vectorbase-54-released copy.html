New features in {{project}} 54

  • We have a new format for the genome, transcriptome (RNAseq and microarray) and proteome (mass spectrometry) data set types. These data pages have been updated to increase clarity and reduce redundancy of information. The genome page is similar to the organisms page we used to have in legacy VectorBase. Follow this link ({{project}} genome data sets) which has all VectorBase (~500) data sets, filtered to show only the genomes. For example in the genome page, a new table "Explore other Categories of Data Associated with this Genome" contains links to all functional data sets for the organism. The new "Data Context" section provides counts for genes, assembled sequences and transcripts. 
  • The site search, available in the header, is updated to find versions of an ID (e.g. ID.1, ID.2). Specifically, searching for the ID will retrieve ID, ID.1 and ID.2, etc, while searching for ID.1 will retrieve only ID.1
  • The community annotation software, Apollo, was updated to version 2.6.5 (About Apollo).
  • The BLAT sequence search tool is integrated into Apollo. Use the 'Search' tab from the annotator panel to find your sequence of interest in our Apollo-available genomes. (ApolloBLAT.png)

Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary. Changes in annotation and new experimental data may slightly alter your search results by increasing or decreasing the number of hits. When search parameters change with a new release, we invalidate (Ø) the search and ask you to rerun it. When IDs are updated or removed, we map the old IDs to the new ones, remove the old IDs from your Basket, and leave your Favorites page alone.
