Reference Genome Transition

Recent updates to mosquito and tick species include new assemblies that are much less fragmented, some organized to the chromosome level. All functional data alignments (e.g., RNAseq) will be mapped to both the current and new genomes. After approximately six months, functional data alignments to the old assembly will be removed; old assemblies will remain available on VectorBase indefinitely but with limited functionality. The new assembly will have all the omics data sets aligned and complete site functionality. A banner on the genome page of the species involved will indicate such changes (e.g., Ixodes scapularis PalLabHiFi). If you have any feedback or questions, need help, or are not able to find your genes of interest in the new genomes, please do not hesitate to contact us https://vectorbase.org/vectorbase/app/contact-us.

New data in {{project}} 66

Genome Sequence and Annotation

The following organisms are represented in VectorBase for the first time.
  • Lutzomyia longipalpis M1 – A vector of Leishmania parasites that cause visceral leishmaniasis. A single male (M1) specimen of Lutzomyia longipalpis, derived from the Jacobina strain was sequenced using PacBio HiFi and Illumina HiC technologies. GCF_024334085.1; Labb√© et al. 2023
  • Phlebotomus papatasi M1 – A vector of Leishmania parasites that cause cutaneous leishmaniasis. The genome of individual one male (M1) from the PPIS colony was sequenced by PacBio HiFi using the PacBio Ultralow input library prep, and additional material from the PPIS colony was used to generate HiC data for scaffolding the reads. GCF_024763615.1.
  • Anopheles marshallii AmarGA1 – This assembly (idAnoMarsDA_429_01) is based on 51x PacBio HiFi data, 10X Genomics Chromium data, and Arima Hi-C data generated by the Anopheles Reference Genomes project. GCF_943734725.1.
  • Sabethes cyaneus ScyaPA1 – This assembly, idSabCyanKW18_F2, is based on 37x PacBio HiFi data and Arima Hi-C data generated by the Anopheles Reference Genomes project. GCF_943734655.1.
  • Anopheles nili AnilCM1 – This assembly, idAnoNiliSN_F5_01, is based on 48x PacBio HiFi data, 10X Genomics Chromium data, and Arima Hi-C data generated by the Anopheles Reference Genomes project. GCF_943737925.1.
  • Biomphalaria glabrata XG47 – This assembly, xgBioGlab47.1, is based on 29x PacBio data and Arima2 Hi-C data generated by the Wellcome Sanger Institute Tree of Life Programme. GCF_947242115.1.

Genome Updates

  • Glossina fuscipes IAEA 2018 – This data set was updated with the 2022 annotation version from the INSDC repository. GCA_014805625.1; Saarman et al. 2022 (preprint)

Genome Reference and Strain Name Changes

  • Lutzomyia longipalpis M1 now serves as the VectorBase reference for Lutzomyia species. All functional genomics data will be aligned to this genome.
  • Anopheles darlingi AdarGF1 – The strain name was changed from idAnoDarlMG-H_01 to AdarGF1 beginning with VectorBase 66. This organism now serves as the VectorBase reference for Lutzomyia species. All functional genomics data will be aligned to this genome.
  • Phlebotomus papatasi M1 now serves as the VectorBase reference for Phlebotomus species. All functional genomics data will be aligned to this genome.

Annotation Updates

  • Haemaphysalis longicornis HaeL-2018 – Pfam domain names were substituted for product descriptions for genes with assigned Pfam domains and uninformative product descriptions such as 'unspecified product'. >12,000 genes were given new product descriptions.

RNA Sequence

  • Aedes aegypti female midgut 72-h bloodmeal digestion cycle for 11-time points – Female midgut 72-h bloodmeal digestion cycle for 11-time points, pre- and post-bloodmeal, focusing on the first 24 h. Four (4) independent replicates. Tsujimoto and Adelman 2023.
  • Haemaphysalis longicornis Engorged female from a stray domestic dog – Transcriptome of an engorged female from a stray domestic dog without owner, found in Pingliang, Gansu, China. Niu et al. 2019.
