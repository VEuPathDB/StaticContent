New features in {{project}} 61

Associated Data Sets

  • Data sets that come from the same publication are now linked. The new 'Associated Data Sets' section on any data set record page will link to other data from the same publication within {{project}} as well as data from organisms supported in different VEuPathDB sites. View tutorial GIF.

New data in {{project}} 61

Genome Updates:

  • Cimex lectularius Harlan – updated to the current INSDC version to avoid discrepancies between public versions. GCA_000648675.3
    • .

Genome Sequence and Annotation

  • Anopheles coluzzii AcolN3 idAnoColuKW18-c001_1 – Based on 59x PacBio HiFi data, 10X Genomics Chromium data, and Arima Hi-C data, this sequence is assembled into chromosome-scale scaffolds. Unpublished. GCF_943734685.1 from the Anopheles Reference Genomes Project
  • Anopheles darlingi idAnoDarlMG-H_01 – Based on 32x PacBio HiFi data, 10X Genomics Chromium data, and Arima Hi-C data, this sequence is assembled into chromosome-scale scaffolds. Unpublished. GCF_943734745.1 from the Anopheles Reference Genomes Project
  • Anopheles funestus idAnoFuneDA-416_04 – Based on 56x PacBio HiFi data, 10X Genomics Chromium data, and Arima Hi-C data, this sequence is assembled into chromosome-scale scaffolds. Unpublished. GCF_943734845.2 from the Anopheles Reference Genomes Project

Proteomics

  • Proteome of body with no head, wings, or legs – Proteomic analyses were done with a single pool from all collection time points from lab colony of An. stephensi mosquitoes. Imrie et al. 2019

RNA Sequence

  • Glossina fuscipes reproductive tissues infection with endosymbiotic Spiroplasma – Field captured (Uganda) Spiroplasma infected and uninfected male (testes, accessory glands, and ejaculatory duct) and female (ovaries, spermathecae, uterus, and oocysts) reproductive tissues. Son et al. 2021

Structure Predictions

  • AlphaFold Predictions – are updated to AlphaFold Release 4 (July 2022) which offers predictions for significantly more proteins than Release 3. Use {{project}}'s Genes with AlphaFold Predictions search to explore genes with AlphaFold predictions or visit the Structure Analysis section of your favorite gene page to view data relevant to you.
